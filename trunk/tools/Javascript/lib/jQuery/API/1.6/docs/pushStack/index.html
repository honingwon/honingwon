.pushStack()

Categories: Plugin Authoring

.pushStack( elements ) 返回： jQuery

描述： 将一个DOM元素集合加入到jQuery栈。

  • version added: 1.0.pushStack( elements )

    elements将要压入jQuery栈的元素，用于生成一个新的jQuery对象。

  • version added: 1.3.pushStack( elements, name, arguments )

    elements将要压入jQuery栈的元素，用于生成一个新的jQuery对象。

    name将要压入jQuery栈的元素，用于生成一个新的jQuery对象。

    arguments传递给jQuery方法的参数(用于序列化)。

Example:

Add some elements onto the jQuery stack, then pop back off again.

jQuery([])
    .pushStack( document.getElementsByTagName("div") )
        .remove()
    .end();

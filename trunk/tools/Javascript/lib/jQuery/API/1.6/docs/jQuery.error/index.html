jQuery.error

Categories: Plugin Authoring

jQuery.error( message )

描述： 接受一个字符串，并抛出包含这个字符串的异常。

  • version added: 1.4.1jQuery.error( message )

    message将要发出的消息.

这个方法主要是为了让插件开发人员可以重载此方法，并以更好的方式显示错误消息，或者提供更多相关信息。

Example:

重载 jQuery.error 以便可以在 Firebug 中显示。

jQuery.error = console.error;

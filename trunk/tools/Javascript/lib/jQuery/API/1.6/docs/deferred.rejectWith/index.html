deferred.rejectWith()

Categories: Deferred Object

deferred.rejectWith( context, [ args ] ) 返回： Deferred

描述： 拒绝延迟的对象，并根据给定的上下文和参数调用任何失败的回调函数。

  • version added: 1.5deferred.rejectWith( context, [ args ] )

    context 上下文作为this对象传递给failCallbacks 。

    args 传递给failCallbacks的可选参数。

当延迟被拒绝，任何failCallbacks添加的deferred.then或deferred.fail被调用。回调按他们添加的顺序执行。每个回调传递的args在deferred.reject()中调用。之后添加任何failCallbacks递延被拒绝进入状态时，立即执行添加，使用的参数被传递给.reject()调用。有关详细信息，请参阅文件Deferred object 。

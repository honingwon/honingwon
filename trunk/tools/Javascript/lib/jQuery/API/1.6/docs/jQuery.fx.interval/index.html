jQuery.fx.interval

Categories: Effects > Custom | Properties > Properties of the Global jQuery Object

jQuery.fx.interval 返回： Number

描述： 该动画的频率（以毫秒为单位）。

  • version added: 1.4.3jQuery.fx.interval

这个属性可以被操作，以调整它的动画每秒运行帧数。默认是13毫秒。修改为一个较小的数字可能使动画在更快浏览器（如Chrome）中运行更流畅，但这样做有可能影响性能。

Example:

为所有的动画运行较少的帧。

<!DOCTYPE html>
<html>
<head>
  <style>
    div { width:50px; height:30px; margin:5px; float:left;
          background:green; }
    </style>
  <script src="http://code.jquery.com/jquery-latest.min.js"></script>
</head>
<body>
  <p><input type="button" value="Run"/></p>
<div></div>
<script>
jQuery.fx.interval = 100;

$("input").click(function(){
  $("div").toggle( 3000 );
});
  </script>

</body>
</html>

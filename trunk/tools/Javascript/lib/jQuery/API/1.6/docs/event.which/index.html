event.which

Categories: Events > Event Object

event.which 返回： String

描述： 针对键盘和鼠标事件，这个属性能确定你到底按的是哪个键。

  • version added: 1.1.3event.which

event.which 将 event.keyCode 和 event.charCode 标准化了。推荐用 event.which 来监视键盘输入。更多细节请参阅： event.charCode on the MDC.

Example:

记录按键

<!DOCTYPE html>
<html>
<head>
  <script src="http://code.jquery.com/jquery-latest.min.js"></script>
</head>
<body>
  
<input id="whichkey" value="type something">
<div id="log"></div>
<script>$('#whichkey').bind('keydown',function(e){ 
  $('#log').html(e.type + ': ' +  e.which );
});  </script>

</body>
</html>

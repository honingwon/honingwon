event.currentTarget

Categories: Events > Event Object

event.currentTarget 返回： Element

描述： 在事件冒泡阶段中的当前DOM元素

  • version added: 1.3event.currentTarget

这个属性总是等于this函数。

Example:

警报currentTarget的匹配`this`关键字。

$("p").click(function(event) {
  alert( event.currentTarget === this ); // true
});
